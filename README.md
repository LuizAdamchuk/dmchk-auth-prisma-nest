## Auth Setup

This is an example of how to implement an authentication system in NestJs using passport.js, and json web tokens (JWT).

I've included integration tests and e2e tests.

## Techs

- TS
- Prisma
- Passport JWT
- NestJs
- Argon2
- Jest
- Supertest

### Starting

```
npm i
```

```
npm run test:int
```

```
docker-compose run
```

```
npx prisma studio
```
